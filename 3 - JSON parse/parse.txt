
O método JSON.parse() analisa uma string JSON, construindo o valor ou um objeto JavaScript descrito pela string. 
Uma função reviver opcional pode ser fornecida para executar uma transformação no objeto que será retornado.

Sintaxe

JSON.parse(text[, reviver])

Parâmetros

text
A string para analisar como JSON. Veja o objeto JSON para uma descrição da sintaxe JSON.

reviver
Se for uma função, prescreve como o valor originalmente produzido pela análise será transformado antes de ser retornado.
